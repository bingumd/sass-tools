# @bingumd/sass-tools

Sass plugin to add a collection of mixins, helpers and tools for your project.

## :hammer: Installation

##### Yarn
```sh
yarn add @bingumd/sass-tools
```

##### Npm
```sh
yarn install @bingumd/sass-tools
```

## :gear: Usage

```scss
@use '@bingumd/sass-tools' as util;
```
